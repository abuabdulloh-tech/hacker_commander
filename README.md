# CYBERCOMMANDER

A hacker-themed, localized, dual-pane portable file manager built in Python using PyQt6.
Operatsion tizim tiliga moslashuvchi, kiber-xaker mavzusidagi ikki panelli portativ fayl menejeri.
Локализованный двухпанельный портативный файловый менеджер в стиле хакера на PyQt6.

---

## 🇺🇸 ENGLISH

### Key Features
- **Dual-Pane Interface**: Classic Total Commander split layout with multiple active directory tabs (`Ctrl+T`).
- **Drag & Drop**: Copy files instantly by dragging them between panels or dropping from outside.
- **Ctrl+Z Undo**: Rolling history stack to cancel copy, move, folder creation, and renaming operations.
- **Built-in Lister & Editor**: Hex view binary reader (`F3`) and raw text editor (`F4` / `Shift+F4`).
- **Security Utilities**: Secure file shredder (`F8` / `Shift+Delete` 3-pass overwrite) and hash generator (`F4` Edit -> Analyzer).
- **History Tracker**: Navigate directory paths via history keys (`Alt+Left` / `Alt+Right` / `Alt+Down`).
- **Cyber Console**: Interactive terminal with simulated hacker scripts (`matrix`, `scan`, `decrypt`) or native shell redirection.

### Mapped Controls
- `F1`: Help Guide
- `F2`: Reread / Refresh active folder
- `F3`: Hex Viewer (Lister)
- `F4`: built-in Text Editor
- `F5`: Copy selected items
- `F6`: Move selected items
- `F7`: New Folder (Mkdir)
- `F8` / `Delete`: Delete permanently (Move to trash / Shred)
- `Alt+F1` / `Alt+F2`: Change Left / Right active drive
- `Alt+Enter`: Metadata Properties
- `Ctrl+A`: Select All
- `Ctrl+B`: Flat list view (Recursive files list)
- `Ctrl+D`: Bookmark Hotlist
- `Ctrl+M`: Multi-Rename Tool
- `Ctrl+Q`: Toggle live preview on the opposite panel
- `Ctrl+H`: Toggle hidden files

## 🇷🇺 РУССКИЙ

### Основные Возможности
- **Двухпанельный интерфейс**: Классический сплит-лейаут с поддержкой нескольких вкладок каталогов (`Ctrl+T`).
- **Drag & Drop**: Быстрое копирование файлов путем перетаскивания между панелями или из проводника.
- **Ctrl+Z Отмена**: Стек истории для отмены операций копирования, перемещения, создания папок и переименования.
- **Встроенный редактор**: Hex-просмотрщик бинарных данных (`F3`) и текстовый редактор (`F4` / `Shift+F4`).
- **Утилиты безопасности**: Безвозвратное затирание файлов шредером (`F8` / `Shift+Delete`) и анализатор хэш-сумм.
- **История путей**: Перемещение по истории переходов (`Alt+Left` / `Alt+Right` / `Alt+Down`).
- **Кибер-консоль**: Терминал с интерактивными скриптами-симуляторами (`matrix`, `scan`, `decrypt`) и системной командной строкой.

### Управляющие Клавиши
- `F1`: Руководство по командам
- `F2`: Обновить активный каталог
- `F3`: Просмотр в Hex-виде
- `F4`: Встроенный редактор текста
- `F5`: Копировать выбранные элементы
- `F6`: Переместить выбранные элементы
- `F7`: Создать каталог (Mkdir)
- `F8` / `Delete`: Безопасное уничтожение / Удаление
- `Alt+F1` / `Alt+F2`: Смена диска Левой / Правой панели
- `Alt+Enter`: Свойства метаданных
- `Ctrl+A`: Выбрать все
- `Ctrl+B`: Плоский список файлов (Без учета структуры папок)
- `Ctrl+D`: Горячий список закладок
- `Ctrl+M`: Групповое переименование
- `Ctrl+Q`: Включить быстрый просмотр файла на соседней панели
- `Ctrl+H`: Показать/скрыть скрытые файлы
